## Installation

```
npm install --save covfefe
```

## Usage

### Usage -- Library

```javascript
> const covfefe = require('covfefe');

> covfefe('Despite the constant negative press');
"Despite the constant negative press covfefe"
```

### Usage -- CLI

```
$ covfefe "despite all my rage"
despite all my rage covfefe
```
